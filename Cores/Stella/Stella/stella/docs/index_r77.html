Quick Navigation Guide


Stella for RetroN 77 can be controlled via the console buttons or a joystick in the left or right port. The resulting actions are depending on the current mode and are described below.

During Emulation

The joysticks work normal and all console buttons as labeled except of the following.

Joystick Button Action
Button 4 4:3,16:9 Open the Command Menu
Button 5 FRY Return to launcher
Button 6 - Open settings
Button 7 - Rewind game
Button 8 MODE Select
Button 9 RESET Reset

In the Launcher

Joystick Button Action
Up SAVE Previous game
Down RESET Next game
Left LOAD Page up
Right MODE Page down
Button 1 SKILL P1 Start selected game
Hold Button 1 - Open a context menu with the following choices:
  • Add to favorites
  • Power-On options...
  • Go to initial directory
  • Go to parent directory
  • Settings...
Button 2 or 6 close the menu without any action.
Button 2 SKILL P2 Open Power-On options
Button 4 COLOR, B/W Open the Settings Dialog

In a Dialog

Joystick Button Action
Up SAVE Increase current setting
Down RESET Decrease current setting
Left LOAD Previous dialog element
Right MODE Next dialog element
Button 1 SKILL P1 Select element
Button 2 SKILL P2 OK
Button 3 or
Button 1 + Left		 4:3,16:9 Previous tab
Button 4 or
Button 1 + Right		 FRY Next settings
Button 6 - Cancel

Note: If you have a keyboard attached via OTG, all keys work as described in the Stella User's Guide.

Two dialogs were especially (re)designed for the RetroN 77:

Command Menu

This dialog is similar to the PC version's Command Menu, but with some commands especially selected for the RetroN 77.

Basic Settings Dialog

'Use Advanced Settings...' switches settings mode. Then you have access to all Stella setting. But please be careful, make sure that you know what you are doing!
The 'Help...' dialog explains the function of the console and controller buttons, similar to this document.
Finally the dialog combines the most important settings from several other dialogs of the PC version:
  • For 'UI theme' and 'Dialogs position' see User Interface Settings - Look & Feel.

  • 'TV mode', 'Scanline intensity', 'Phosphor blend' and 'Overscan' are described in Video & Audio Settings - Display and TV Effects.

  • 'Left port' and 'Right port' are defined per game. For details see Game Properties - Controller Properties.
